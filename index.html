What is Spark?

SPARK is a welcoming program by the Open Source Software Club designed specifically for first-year students. It's your gateway to the tech world, requiring no prior experienceâ€”just curiosity and enthusiasm to learn web development fundamentals.

Do I need any prior knowledge to attend SPARK classes?

No prior knowledge is required! SPARK is designed for complete beginners. Whether you've never written a line of code or are curious about web development, our program starts from the basics and builds your skills step by step.

Will I get any practical experience or just theory?

You'll get hands-on practical experience! Our program focuses on learning by doingâ€”you'll write actual code, build real web pages, work with HTML, CSS, and JavaScript, and use Git to manage projects. Theory is taught through practical application.

Is SPARK only for CS/IT students?

Not at all! SPARK welcomes students from all academic backgrounds. Whether you're studying Computer Science, Arts, Commerce, Engineering, or any other field, if you're curious about web development, you're welcome to join our program.

What will I be able to do after the classes are completed?

After completing SPARK, you'll be able to create responsive websites using HTML, CSS, and JavaScript. You'll understand version control with Git, have a portfolio of projects, and possess the foundational skills to pursue advanced web development or join the Open Source Software Club.
